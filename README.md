# Data Analysis Practice

This repo is where I’m working on improving my data analysis skills through small projects using Python, pandas, and SQL.

The goal isn’t just to complete projects, but to get better at the full process — cleaning data, validating it, analyzing trends, and explaining results clearly.

## What’s in here

Each folder is a separate project. Some are more complete than others since this repo also tracks my learning process.

Projects may include:
- Data cleaning and validation
- Exploratory data analysis (EDA)
- Working with real-world messy datasets
- Basic visualizations
- SQL queries and data extraction

## Tools I’m using

- Python
- pandas
- SQL

## Notes

Some projects might be incomplete or pivoted (for example, if the dataset didn’t support the original question). I keep them anyway as part of the learning process and to show how I approach problems and adjust.

## Goal

To build stronger, more consistent data analysis skills over time and get comfortable working with real datasets.
